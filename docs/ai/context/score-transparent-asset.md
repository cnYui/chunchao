# 琴谱素材透明背景处理

## 目标

为 `pic/琴谱.png` 去除纯白和近白背景，输出可直接叠加到前端第二页背景上的透明 PNG。

## 输入文件

- `pic/琴谱.png`

## 输出文件

- `pic/琴谱_transparent.png`
- 前端当前实际使用其 ASCII 别名 `pic/score-transparent.png`

## 处理策略

- 读取原图并按亮度计算每个像素的透明度
- 将近白背景映射为透明，将深色谱面内容保留为不透明
- 输出像素统一写为黑色，原因是目标内容本身为黑色谱面，避免原图压缩产生的灰边和彩色杂质叠到页面上
- 采用分段线性阈值保留边缘过渡，原因是直接二值化容易削掉细线和抗锯齿边缘
- 对过低透明度的残留直接清零，原因是当前原图右中区域存在压缩污迹，低 alpha 残留叠到背景上仍会显脏

## 当前阈值

- 亮度小于等于 `180` 时，alpha 设为 `255`
- 亮度大于等于 `245` 时，alpha 设为 `0`
- 中间区间按线性插值计算 alpha
- 计算后若 `alpha < 128`，直接清零

## 校验结果

- 输出文件格式为 `RGBA PNG`
- 输出尺寸保持 `2888 x 1440`
- 输出 alpha 范围为 `0 ~ 255`
- 原图未覆盖
