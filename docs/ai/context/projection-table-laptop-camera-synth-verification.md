# 桌面投影合成器现场验证

## 启动前

- 内置摄像头能拍到整张投影桌面
- 页面通过 `localhost` 或 `https` 打开
- 16 个格子初始为空

## 标定

- 在右下角摄像头预览中按 `左上 -> 右上 -> 右下 -> 左下` 点击 4 个角点
- 标定完成后，调试层中的 16 个 ROI 与实际投影格子对齐
- 若窗口尺寸变化或投影位置变化，重新标定
- 运行态 pad 亮起或熄灭时，调试层 ROI 不应随按钮高亮发生位置偏移

## 占格

- 点击“采集空场”前，16 格内没有方块和手
- 任一格放入方块后，`300-500ms` 内开始发声
- 任一格移开方块后，`300-500ms` 内停止发声
- 手经过格子上方时，不应频繁误触发占格
- 倾斜视角下，相邻格边缘不应因为 ROI 包围盒重叠而串格触发

## 手部

- 手指进入 `VIBE` 区域后，可稳定切换音色方向
- 手指进入 `VOL / REV / POS` 区域后，可稳定调节对应参数
- 鼠标仍可作为后备输入调试旋钮、滑杆和 pad

## 调试层

- 摄像头预览可见
- 四个标定点可见
- 16 个 ROI 边框可见
- baseline 状态可见
